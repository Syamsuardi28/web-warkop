Panduan singkat:

1. Ganti gambar di folder `assets/`.
2. Buka `index.html` di browser atau jalankan local server.
3. Untuk menyimpan pesan, tambahkan endpoint backend pada bagian form di `js/main.js`.

Fitur yang disertakan:

- Struktur HTML5 semantik
- CSS responsif (Grid & Flexbox)
- Animasi steam CSS
- Hover effects pada kartu
- Filter menu, carousel sederhana, dark mode, validasi form
- Scroll reveal ringan tanpa library eksternal
